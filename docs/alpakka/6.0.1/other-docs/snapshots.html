Snapshots

Snapshots are published to the Sonatype Snapshot repository after every successful build on master. Add the following to your project build definition to resolve Alpakka snapshots:

Configure repository

Maven
<project>
...
  <repositories>
    <repository>
        <id>snapshots-repo</id>
        <name>Sonatype snapshots</name>
        <url>https://oss.sonatype.org/content/repositories/snapshots</url>
    </repository>
  </repositories>
...
</project>
sbt
resolvers += Resolver.sonatypeRepo("snapshots")
Gradle
repositories {
  maven {
    url  "https://oss.sonatype.org/content/repositories/snapshots"
  }
}

Documentation

The snapshot documentation is updated with every snapshot build.
