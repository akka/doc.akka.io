Certified Java versions

Java 21

We are currently in the process of certifying Temurin 21 builds for use with Akka.

Java 17

Temurin 17 builds and Red Hat Enterprise Linux builds of OpenJDK 17 are certified for use with some components of the Lightbend Platform.

Java 11

Temurin 11 builds and Red Hat Enterprise Linux builds of OpenJDK 11 are certified for use with some components of the Lightbend Platform.

Java 8

Starting with the Akka 23.10 releases, we do not provide Java 8 compatible builds of Lightbend technologies anymore.

Java Platform Module System (JPMS)

Use of the Java Platform Module System (JPMS) is Not Certified at this time.
