Akka Enhancements

Akka Enhancements is a suite of useful components that complement Akka, available as part of the Lightbend Platform Subscription. It includes:

  • Akka Resilience Enhancements
    • Akka Split Brain Resolver
    • Kubernetes Lease
    • Akka Thread Starvation Detector
    • Akka Config Checker
    • Akka Diagnostics Recorder
    • Akka Resilience Enhancements Release Notes
  • Akka Persistence Enhancements
    • Akka Multi-DC Persistence
    • GDPR for Akka Persistence
    • Akka Persistence Enhancements Release Notes

See also:

  • Lightbend Telemetry - distributed tracing, metrics and monitoring for Akka Actors, Cluster, HTTP as well as Scala Futures and more
