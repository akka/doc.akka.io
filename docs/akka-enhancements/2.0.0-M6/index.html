Akka Enhancements

Akka Enhancements is a suite of components that complement Akka, available to licensed customers.

These contain additional connectors align with the Alpakka project for stream-aware and reactive integration pipelines for Java and Scala.

Versions

The code in this documentation is compiled against:

  • Akka Enhancements 2.0.0-M6
  • Scala 2.13
  • Akka Streams 2.8.4+ (Reference, Github)
