akka.actor.typed

ExtensionSetup 

abstract class ExtensionSetup[T <: Extension] extends Setup

Each extension typically provide a concrete ExtensionSetup that can be used in akka.actor.setup.ActorSystemSetup when starting the ActorSystem to replace the default implementation of the extension. Intended for tests that need to replace extension with stub/mock implementations.
