final case class MalformedQueryParamRejection(parameterName: String, errorMsg: String, cause: Option[Throwable] = None) extends javadsl.server.MalformedQueryParamRejection with RejectionWithOptionalCause with Product with Serializable

Rejection created by parameter filters. Signals that the request was rejected because a query parameter could not be interpreted.
