Companion object HttpCharsetRange

sealed abstract class HttpCharsetRange extends javadsl.model.HttpCharsetRange with ValueRenderable with WithQValue[HttpCharsetRange]

A charset range as encountered in Accept-Charset. Can either be a single charset, or * if all charsets are supported and optionally a qValue for selecting this choice.
