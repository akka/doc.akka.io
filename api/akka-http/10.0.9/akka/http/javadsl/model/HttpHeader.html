akka.http.javadsl.model

HttpHeader 

abstract class HttpHeader extends AnyRef

The base type representing Http headers. All actual header values will be instances of one of the subtypes defined in the headers packages. Unknown headers will be subtypes of akka.http.javadsl.model.headers.RawHeader.
