    akka.persistence.update.javadsl.JournalTransformer implementation for JDBC.

    It is retrieved with:

    JournalTransformer transformer =
  JournalUpdater.get(system).getJournalUpdaterFor(JdbcJournalTransformer.class, JdbcJournalTransformer.Identifier());

    Corresponding Scala API is in akka.persistence.update.jdbc.scaladsl.JdbcJournalTransformer.

    Configuration settings can be defined in the configuration section with the absolute path corresponding to the identifier, which is "jdbc-journal-updater" for the default JdbcJournalTransformer#Identifier. See reference.conf.
