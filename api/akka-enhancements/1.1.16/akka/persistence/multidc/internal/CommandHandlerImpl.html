CommandHandlerImpl 

class CommandHandlerImpl[Command, Event, State] extends CommandHandler[Command, Event, State] with CommandHandler[Command, Event, State]

Command handlers and optional functions for other signals, e.g. Termination messages if watch is used. CommandHandlerImpl is an immutable class.
