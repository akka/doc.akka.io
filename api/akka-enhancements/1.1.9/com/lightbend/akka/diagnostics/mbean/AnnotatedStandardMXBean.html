AnnotatedStandardMXBean 

class AnnotatedStandardMXBean extends StandardMBean

INTERNAL API

Copied from http://actimem.com/java/jmx-annotations/ which source code is licensed under the Apache License, Version 2.0.

The StandardMBean falls short with providing meaningful descriptions and parameter names. This class reads those things from Description and Name annotations of the MBean interface class and methods.

Usage: mbeanServer.registerMBean(new AnnotatedStandardMXBean(mbeanImpl, SomeMBean.class), name);
