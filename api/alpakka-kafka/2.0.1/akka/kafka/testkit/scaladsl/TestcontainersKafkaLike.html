TestcontainersKafkaLike

trait TestcontainersKafkaLike extends KafkaSpec with Spec

Uses Testcontainers to start a Kafka cluster in a Docker container. This trait will start Kafka only once per test session. To create a Kafka cluster per test class see TestcontainersKafkaPerClassLike.

The Testcontainers dependency has to be added explicitly.
