  6. def commitableSink[K, V](settings: ProducerSettings[K, V], producer: Producer[K, V]): Sink[Message[K, V, Committable], CompletionStage[Done]]

    Sink that is aware of the committable offset from a Consumer.

    Sink that is aware of the committable offset from a Consumer. It will commit the consumer offset when the message has been published successfully to the topic.

    Note that there is always a risk that something fails after publishing but before committing, so it is "at-least once delivery" semantics.
